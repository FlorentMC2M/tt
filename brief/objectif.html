Test technique

 

Objectif

Présenter une liste de rubriques sous forme d'arborescence, avec la possibilité d'éditer chacune d'entre elles et de les trier entre elles.

Les différentes parties du test peuvent être réalisées par ordre de préférence.

Environnement

·         Afin de connaître l'URL où est accessible votre projet, tapez cette commande dans la console bash de Cloud9 :

echo http://$C9_HOSTNAME

·         Les informations concernant la base de données sont les suivantes :

o   Type : MySQL

o   Nom de la base de données : c9

o   Nom de la table : sections

 

o   Pour connaître l'adresse IP de la base de données, tapez cette commande dans la console bash de Cloud9 :

echo $IP

o   Pour connaître le nom d'utilisateur de la base de données, tapez cette commande dans la console bash de Cloud9 :

echo $C9_USER
 

o   Il n'y a pas de mot de passe

o   Vous devez importer les données avec les commandes suivantes dans la console bash de Cloud9 :

cd ~/workspace
mysql-ctl cli
use c9
source sections.sql

Intégrer la liste des rubriques

Cela consiste réaliser l'intégration de la page afin d'obtenir un résultat similaire à celui présenté dans la maquette  brief/liste-des-rubriques.jpg

L'entête et le pied de page sont à intégrer dynamiquement à partir des URL suivantes :

·         http://tools.mc2m.fr/test_technique/header.html

·         http://tools.mc2m.fr/test_technique/footer.html

Les polices sont présentes dans le répertoire font.

Créer une page d'édition d'une rubrique

Cette page doit présenter un formulaire permettant de modifier le titre de la rubrique et de lui associer une image par téléchargement.

Seuls les formats d'image JPEG et PNG doivent être autorisés. L'image une fois téléchargée doit être accessible par URL.

Une fois le formulaire envoyé, un message doit être affiché à l'utilisateur pour l'informer du succès ou de l'échec de sa requête.

Un échantillon d'images est disponible dans le répertoire brief/images.

Permettre le tri des rubriques au sein de leur rubrique mère

Dans la page présentant la liste des rubriques, il faut donner la possibilité de trier les rubriques entres elles au sein de leur rubrique mère, à l'aide d'un composant de glisser-déposer en JavaScript.

Le nouvel ordre doit être sauvegardé en base de données.
